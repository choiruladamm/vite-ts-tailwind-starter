
3. Deploy to Vercel

   ![Deploy with Vercel](https://vercel.com/button)

### 2. Install dependencies

It is encouraged to use **npm** so the husky hooks can work properly.

```bash
npm install
```

### 3. Run the development server

You can start the server using this command:

```bash
npm run dev
```
